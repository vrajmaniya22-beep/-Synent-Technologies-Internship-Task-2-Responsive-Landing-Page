Features

Built for modern experiences

Every detail designed to deliver a premium, buttery-smooth navigation experience.

Scroll Aware

Navbar morphs on scroll — transparent hero view transitions to a frosted glass surface.

Fully Responsive

Seamlessly adapts from ultra-wide desktops to compact mobile screens with elegant breakpoints.

Hamburger Toggle

Animated hamburger-to-X morph with a full-screen overlay menu featuring staggered reveals.

60fps Smooth

GPU-accelerated transitions and CSS-only animations ensure silky performance everywhere.

Active Indicator

A sliding pill indicator smoothly glides to highlight the current section as you navigate.

Accessible

Keyboard navigable, ARIA-labeled, and screen-reader friendly by default.

About

Crafted with obsessive attention to detail

This responsive navigation bar is built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies. Every transition, every animation, every pixel is deliberately designed to feel premium and alive.

  • Zero dependencies — pure vanilla stack
  • Glassmorphism on scroll with backdrop-filter
  • CSS custom properties for theming
  • Under 10KB total — blazing fast load
